¿Qué es python?

Python es un lenguaje de programación interpretado, es decir, no es necesario traducirlo a lenguaje de máquina para que esté ejecute las funciones dentro del código que has desarrollado. Es conocido como un lenguaje de programación orientado a objeto. Se trata de un lenguaje de programación multiparadigma, ya que soporta orientación a objetos, programación imperativa y, en menor medida, programación funcional.

Se distribuye bajo la licencia de código abierto PSF y cuenta con un modelo de desarrollo comunitario, en el que cualquiera puede aportar extensiones al núcleo del lenguaje. Si algo caracteriza a este lenguaje programación es la comunidad a la que pertenece, es un lenguaje donde puedes encontrar información muy útil y de manera muy fácil, ya que las personas que lo saben como funciona suelen compartir los conocimientos de este.

Mi recomendación en cuanto a si Python es el primer lenguaje de programación que deberías, es simple. Haz lo que pienses que te puede hacer feliz y con lo que realmente te sientas cómodo.

Este es uno de los fuertes de Python, tiene como filosofía la legibilidad de su código, y es ahí donde reside su facilidad para ser aprendido. Es un lenguaje de programación que se parece mucho al lenguaje humano.

Mi experiencia personal con Python ha sido muy buena, debido a que en muchos casos escribes código como hablas. Es un lenguaje muy dinámico y el aprendizaje es vertiginosamente rápido, después claro está, siempre puedes aprender cosas nuevas como todo en la vida.

Para mí Python es, sin lugar a dudas, el lenguaje de futuro. Qué en completo con otros, puedes lograr proyectos tan ambiciosos como es Netflix.
